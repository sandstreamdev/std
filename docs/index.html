merge

Merges two objects deeply.

Type signature

Examples

merge({ a: 1, b: 3 }, { b: 7 }); // ⇒ { a: 1, b: 7 }
merge({ a: 1, b: 3 }, { b: { d: 8 } }); // ⇒ { a: 1, b: { d: 8 } }
