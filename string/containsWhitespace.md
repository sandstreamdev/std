# containsWhitespace

Checks if the given string contains whitespace.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(x: string) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
containsWhitespace("test string");
// ⇒ true
```

```javascript
containsWhitespace("test");
// ⇒ false
```
