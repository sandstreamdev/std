any

Checks if the given array is not empty (contains at least one element).

Type signature

(xs: any[]) => boolean

Examples

Try in REPL 
any([]); // => false
any([1, 2, 3]); // => true
any([]); // => false
any([1, 2, 3]); // => true
any([]); // => false
any([1, 2, 3]); // => true
