# single

Checks if the given array contains exactly one element.

## Type signature

<!-- prettier-ignore-start -->
```typescript
<T>(xs: T[]) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
single([1]);
// ⇒ true
```

```javascript
single([1, 2, 3]);
// ⇒ false
```

```javascript
single([]);
// ⇒ false
```
<!-- prettier-ignore-end -->
