# differs

Checks if two arrays are not equal.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(xs?: any[], ys?: any[]) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
differs([1, 2, 3], [1, 2]); // ⇒ true
```

```javascript
differs([1, 2, 3], [1, 2, 3]); // ⇒ false
```
<!-- prettier-ignore-end -->
