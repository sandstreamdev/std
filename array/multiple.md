# multiple

Checks if the given array contains more than one element.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(xs: any) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
multiple([1, 2, 3]); // ⇒ true
```

```javascript
multiple([1, 2]); // ⇒ true
```

```javascript
multiple([1]); // ⇒ false
```

```javascript
multiple([]); // ⇒ false
```
<!-- prettier-ignore-end -->
