# last

Returns the last element or undefined when there are no elements in the given array.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(xs: any[]) => any
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
last([1, 2, 3]); // ⇒ 3
```

```javascript
last([]); // ⇒ undefined
```
<!-- prettier-ignore-end -->
