# first

Returns the first element or undefined when there are no elements in the given array.

## Type signature

<!-- prettier-ignore-start -->
```typescript
([x]: [any]) => any
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
first([1, 2, 3]); // ⇒ 1
```

```javascript
first([]); // ⇒ undefined
```
<!-- prettier-ignore-end -->
