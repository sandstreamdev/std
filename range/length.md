# length

Computes the signed length of the given range.

## Type signature

<!-- prettier-ignore-start -->
```typescript
([min, max]: [number, number]) => number
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
length([3, 15]);
// ⇒ 12
```

```javascript
length([1, 0]);
// ⇒ -1
```
<!-- prettier-ignore-end -->
