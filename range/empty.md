# empty

Checks if the given range is empty.

## Type signature

<!-- prettier-ignore-start -->
```typescript
([min, max]: [number, number]) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
empty([2, 2]); // ⇒ true
empty([1, 5]); // ⇒ false
```
<!-- prettier-ignore-end -->
