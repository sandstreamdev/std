# equals

Checks if the given ranges are equal.

## Type signature

<!-- prettier-ignore-start -->
```typescript
([a, b]: [number, number], [c, d]: [number, number]) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
equals([1, 2], [1, 2]);
// ⇒ true
```

```javascript
equals([4, 3], [1, 2]);
// ⇒ false
```
<!-- prettier-ignore-end -->
