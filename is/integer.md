# integer

Checks if the given value is an integer.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(x?: unknown) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript

// ⇒ true
```

```javascript
integer(32.5);
// ⇒ false
```
