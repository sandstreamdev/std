# number

Checks if the given value is a number.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(x?: unknown) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
number(0 / 0);
// ⇒ false
```

```javascript
number(15.6);
// ⇒ true
```
<!-- prettier-ignore-end -->
