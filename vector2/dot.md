# dot

Calculates a dot product of the given vectors. Returns a scalar.

## Type signature

<!-- prettier-ignore-start -->
```typescript
([a, b]: [number, number], [c, d]: [number, number]) => number
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
dot([3, 5], [-1, 8]);
// ⇒ 37
```

```javascript
dot([3, 5], [-1, -8]);
// ⇒ -43
```
<!-- prettier-ignore-end -->
