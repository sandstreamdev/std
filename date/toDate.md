# toDate

Extracts padded YYYY-MM-DD date string out of the given date object.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(date: Date) => string
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
