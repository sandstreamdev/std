# daysInYear

Calculates the number of days in a particular year. Varies by the leap year.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(year: number) => 366 | 365
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
daysInYear(2019);
// ⇒ 365
```

```javascript
daysInYear(2020);
// ⇒ 366
```
<!-- prettier-ignore-end -->
