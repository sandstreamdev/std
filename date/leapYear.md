# leapYear

Detects if a given year is a leap year.

## Type signature

<!-- prettier-ignore-start -->
```typescript
(year: number) => boolean
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
leapYear(2020);
// ⇒ true
```

```javascript
leapYear(2019);
// ⇒ false
```
<!-- prettier-ignore-end -->
