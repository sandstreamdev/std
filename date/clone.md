# clone

Returns the cloned date

## Type signature

<!-- prettier-ignore-start -->
```typescript
(date: Date) => Date
```
<!-- prettier-ignore-end -->

## Examples

<!-- prettier-ignore-start -->
```javascript
clone(new Date('2019-04-24T13:54:33.232Z')); // ⇒ Wed Apr 24 2019 15:54:33 GMT+0200 (czas środkowoeuropejski letni)
```
<!-- prettier-ignore-end -->
